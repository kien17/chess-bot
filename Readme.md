## Requirements

- Tạo môi trường ảo (virtual environment)
```
# Windows
py -m venv venv

# Linux / macOS:
python3 -m venv venv
```

- Kích hoạt môi trường ảo
```
# Windows
venv\Scripts\Activate.ps1

# Linux / macOS:
source venv/bin/activate
```

- Cài đặt các thư viện từ `requirements.txt`
```
pip install -r requirements.txt
